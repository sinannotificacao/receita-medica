# Solicitação SUS - 2 vias lado a lado

Projeto simples em HTML, CSS e JavaScript para preencher uma solicitação uma vez e imprimir duas cópias lado a lado em uma folha A4 na orientação paisagem.

## Estrutura

- `index.html`
- `styles.css`
- `script.js`
- `img/logo.jpg`

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Preencha os campos.
3. Clique em **Imprimir A4**.
4. Na impressão, mantenha **A4** e **paisagem**.

## Publicação no GitHub Pages

Basta subir os arquivos mantendo a pasta `img`.
